# Feishu for Codex v0.1.0

Initial open-source release of `Feishu for Codex`.

## What is included

- Stable local HTTP-backed Feishu MCP server
- Feishu branding assets for plugin identity
- GitHub-ready README in English and Simplified Chinese
- Git marketplace import support through `plugins/feishu`
- Authentication runbook and local doctor script

## Verified capabilities

- `im_v1_chat_list`
  - real-data validation passed
- `docx_builtin_search`
  - real-data validation passed
- `wiki_v1_node_search`
  - real-data validation passed

## Recommended use cases

- Summarize recent Feishu group messages
- Search Feishu Docs and Wiki from Codex
- Draft bot-style replies with Feishu context
- Push recurring Codex project digests to Feishu chats

## Installation

- Source: `https://github.com/hunkwu/plugins-codex-feishu.git`
- Git reference: `main`
- Sparse path: `plugins/feishu`

## Notes

- The stable default path uses the local `lark-mcp` wrapper in `scripts/feishu_http_mcp.py`.
- The upstream official beta MCP server is still included as `lark-mcp-official-beta` for comparison and debugging.
- OAuth authorization codes are single-use and must be exchanged immediately.
